# Inventory & Supply Chain Management Analysis 📦🚚

An interactive **Power BI dashboard** built to analyze key inventory and supply chain performance metrics. This project helps businesses monitor warehouse efficiency, inventory health, transportation costs, sales trends, lead times, and order fulfillment performance using data-driven visualizations.

# 📌 Project Overview

This dashboard provides a comprehensive overview of inventory and supply chain operations, enabling organizations to:

* Monitor warehouse utilization
* Track inventory performance
* Analyze transportation costs
* Measure supplier efficiency
* Identify backorders
* Evaluate lead times
* Monitor sales trends across years
* Compare inventory across regions and product categories

The dashboard is designed for supply chain managers, warehouse managers, business analysts, and decision-makers.

# 📂 Dataset Information

The dataset contains supply chain operational data with the following fields:

| Column                    | Description                   |
| ------------------------- | ----------------------------- |
| Date                      | Transaction Date              |
| Region                    | Sales Region                  |
| Category                  | Product Category              |
| Supplier                  | Supplier Name                 |
| Warehouse                 | Warehouse Location            |
| Order Status              | Fulfilled, Pending, Cancelled |
| Units Sold                | Quantity Sold                 |
| Inventory Level           | Available Inventory           |
| Transportation Cost       | Shipping Cost                 |
| Order Accuracy            | Order Accuracy (%)            |
| Lead Time (Days)          | Delivery Lead Time            |
| Backorder                 | Backorder Status              |
| Cost of Goods Sold (COGS) | Product Cost                  |
| Average Inventory         | Average Inventory Value       |
| Warehouse Capacity        | Total Warehouse Capacity      |

---

# 📈 Dashboard KPIs

### Warehouse Utilization

* Measures warehouse space utilization.
* Helps identify underutilized warehouses.

### Days Sales of Inventory (DSI)

* Indicates how long inventory remains before being sold.
* Lower values generally indicate better inventory movement.

### Inventory Health

* Monitors overall inventory performance.
* Supports inventory optimization decisions.
